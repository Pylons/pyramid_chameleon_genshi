repoze.bfg.chameleon_genshi
===========================

Bindings for Chameleon Genshi support under repoze.bfg.  This package
module lacks its own documentation, but the API is identical to the
one for chameleon.zpt templates as documented in the `BFG docs
<http://bfg.repoze.org/>`_.  Only templating language differs.
